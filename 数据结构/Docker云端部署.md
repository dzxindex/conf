## 3. Docker

- Nginx 配置

```json
server {
    # 设置80 端口
    listen 80;

    location / {
        # 访问/ 的时候等于访问 /usr/share/nginx/html文件夹内
        root /usr/share/nginx/html;
        index index.html;
    }
}
```
