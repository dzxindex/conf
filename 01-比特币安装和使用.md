# 比特币安装和使用

## 1.比特币的安装

```shell
# 运行regtest模式
# 1.修改配置文件
# 修改~/Library/Application Support/Bitcoin/bitcoin.conf文件
testnet=1

# 运行testnet模式
# 2.测试网络 
regtest=1
```



[bitcoin客户端下载地址](https://bitcoin.org/en/download)

- 主网（真实比特币网络）
  - 默认黄颜色
- 真实测试网络（全世界都在使用，需要同步账本：30G左右）
  - 绿色
  - tsetnet=1
- 本地开发网络（用于开发）
  - regtest=1
